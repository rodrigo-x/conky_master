## Installation
- Clone repo `mkdir ~/.conky/nery && git clone https://github.com/rodrigo-x/conky_master.git ~/.conky/nery`
- Install dependencies (see below)

### Dependencies
Required:  
  - curl
  - sed
  - sensors

Weather:
- find your city_id on `wttr.in` and add into `wheater`

Enjoy!
